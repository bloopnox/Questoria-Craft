## 🌸 About The Bot

A premium anime RPG Telegram bot inspired by **Demon Slayer: Kimetsu no Yaiba**.  
Collect characters, battle enemies, buy weapons, unlock mythical units, and rise as the strongest demon slayer.

---

## ✨ Features

- ⚔️ Battle System  
- 🎴 Character Collection  
- 🔥 Mythical & Limited Characters  
- 💰 Economy System  
- 🛒 Shop & Weapons  
- 🎒 Inventory  
- 👑 Premium System  
- 🎁 Daily Rewards  
- 📊 Player Stats & Ranking  

---

## 🤖 Commands

### Basic
```bash
/start
/profile
/daily
/balance
/inventory
```

### Gameplay
```bash
/char
/battle
/buy
/equip
```

### Premium
```bash
/premium
/redeem
```

### Admin
```bash
/addchar
/addcoins
/broadcast
```
